# 宏观数据字典

## 利率

### 联邦基金利率（FEDFUNDS）

| 项目 | 内容 |
|------|------|
| 数据源 | FRED API → `fred_indicators` |
| 含义 | 美联储基准利率，所有利率之锚 |
| 正常区间 | 2-4%（中性利率附近） |
| 看多黄金 | >5%（高利率不可持续，降息预期） |
| 看空黄金 | 上升中（加息周期） |
| 极端值 | >6%或<1%（极端紧缩/宽松） |
| 发布时间 | FOMC会后立即更新 |
| DB列名 | 系列ID=FEDFUNDS |

### 10年期国债收益率（DGS10）

| 项目 | 内容 |
|------|------|
| 数据源 | FRED API |
| 含义 | 市场定价的长期利率预期 |
| 正常区间 | 3-5% |
| 看多黄金 | 下降（宽松预期，机成本降低） |
| 看空黄金 | 上升（利差扩大，资金流出） |
| 极端值 | >6%或<2% |
| 发布时间 | 每日 |
| DB列名 | 系列ID=DGS10 |

### 2年期国债收益率（DGS2）

| 项目 | 内容 |
|------|------|
| 含义 | 短期政策预期，市场对1-2年内利率的定价 |
| 正常区间 | 3-5% |
| 极端值 | 与10年倒挂（<0.5%）→衰退预警 |
| DB列名 | 系列ID=DGS2 |

### 5年期国债收益率（DGS5）

| 项目 | 内容 |
|------|------|
| 含义 | 中期利率锚 |
| 正常区间 | 3-5% |
| DB列名 | 系列ID=DGS5 |
| ⚠️ | 最近无数据，用DGS10替 |

### 10-2年利差（T10Y2Y）

| 项目 | 内容 |
|------|------|
| 含义 | 收益率曲线形状 |
| 正常区间 | 0.5-2.0% |
| 看空 | 倒挂（<0）→衰退信号 |
| 看多 | 陡峭化（>1.5%）→复苏信号 |
| DB列名 | 系列ID=T10Y2Y |

### 1年期国债收益率（DGS1）

| 项目 | 内容 |
|------|------|
| 含义 | 极短期政策预期 |
| DB列名 | 系列ID=DGS1 |

## 通胀

### CPI（CPIAUCSL）

| 项目 | 内容 |
|------|------|
| 数据源 | FRED API |
| 含义 | 消费者物价指数，通胀核心指标 |
| 正常区间 | 2-3% YoY |
| 看多黄金 | >4%（通胀失控，保值需求） |
| 看空黄金 | <2%（通胀低迷） |
| 极端值 | >8%（高通胀） <0%（通缩） |
| 发布时间 | 每月一次（滞后1个月） |
| DB列名 | 系列ID=CPIAUCSL |

### 核心PCE（PCEPILFE）

| 项目 | 内容 |
|------|------|
| 含义 | 美联储最看重的通胀指标（不含食品能源） |
| 正常区间 | 2-3% |
| 极端值 | >4%或<1.5% |
| 关键 | 这是美联储2%目标参考的指标 |
| DB列名 | 系列ID=PCEPILFE |

### 5年盈亏平衡通胀率（T5YIFR）

| 项目 | 内容 |
|------|------|
| 含义 | 市场定价的5年期通胀预期 |
| 正常区间 | 2-3% |
| 看多黄金 | >3%（通胀预期升温） |
| DB列名 | 系列ID=T5YIFR |

### 10年盈亏平衡通胀率（T10YIE）

| 项目 | 内容 |
|------|------|
| 含义 | 市场定价的10年期通胀预期 |
| 正常区间 | 2-3% |
| 看多黄金 | >3%（长期通胀预期脱锚） |
| DB列名 | 系列ID=T10YIE |

## 实际利率

### TIPS 10年期实际利率（DFII10）

| 项目 | 内容 |
|------|------|
| 数据源 | FRED API |
| 含义 | 扣除通胀后的真实利率 |
| 正常区间 | 0-2% |
| **看多黄金** | **下降/负值**（持有黄金无机会成本） |
| **看空黄金** | **上升**（持有黄金成本上升） |
| 极端值 | >3%或<-1% |
| 关键 | 这是黄金最重要的驱动因子 |
| DB列名 | 系列ID=DFII10 |

---

## 就业

### 失业率（UNRATE）

| 项目 | 内容 |
|------|------|
| 含义 | 劳动力市场健康度 |
| 正常区间 | 3.5-5% |
| 看多黄金 | 上升（经济恶化→降息预期） |
| 极端值 | >6%（衰退）或<3%（过热） |
| 发布时间 | 每月第一个周五 |
| DB列名 | 系列ID=UNRATE |

### JOLTS职位空缺（JTSJOL）

| 项目 | 内容 |
|------|------|
| 含义 | 劳动力需求，领先失业率约6个月 |
| 正常区间 | 6000-8000 |
| 看空黄金 | 高位（劳动力市场火热） |
| DB列名 | 系列ID=JTSJOL |

## 美元

### 贸易加权美元指数（DTWEXBGS）

| 项目 | 内容 |
|------|------|
| 数据源 | FRED API |
| 含义 | 美元对一篮子货币的价值 |
| 正常区间 | 100-130 |
| **看多黄金** | **下降**（弱美元→黄金上涨） |
| **看空黄金** | **上升**（强美元→黄金承压） |
| 极端值 | >140或<90 |
| 发布时间 | 每日 |
| DB列名 | 系列ID=DTWEXBGS |

## 情绪

### VIX恐慌指数

| 项目 | 内容 |
|------|------|
| 数据源 | CFTC TFF → `vix_data` |
| 含义 | 市场恐慌程度 |
| 正常区间 | 12-20 |
| 看多黄金 | >25（恐慌→避险） |
| 利空黄金 | <15（风险偏好高） |
| 极端值 | >40（市场危机） |
