# 宏观分析框架

> 宏观是所有品种的背景板，决定大方向，不决定入场点

---

## 第一层：利率环境（最重要）

### 实际利率 = 名义利率 - 通胀预期

| 看 | 怎么算 |
|----|--------|
| 名义利率 | 10年国债 DGS10 |
| 通胀预期 | 10年盈亏平衡 T10YIE |
| 实际利率 | DGS10 - T10YIE ≈ DFII10 |

### 判定

| 实际利率 | 对黄金/商品 | 操作倾向 |
|---------|------------|---------|
| >2.5% | 强烈利空 | 避开贵金属 |
| 1-2.5% | 利空 | 观望 |
| 0-1% | 中性 | 可配置 |
| -1%至0 | 利好 | 增配 |
| <-1% | 强烈利好 | 重仓 |

---

## 第二层：美联储路径

### 看 FedWatch + 收益率曲线

```
FOMC维持概率 >90% + 收益率曲线正常 → 利率稳定，看品种自身驱动
FOMC降息概率 >50% → 重新评估第一层（利率要下）
收益率曲线倒挂（T10Y2Y < 0）→ 衰退预警，利好避险资产
```

---

## 第三层：美元方向

### 对商品的影响

| 美元 DXY | 对商品 |
|---------|--------|
| 大幅走强 | 全品种承压 |
| 稳步走弱 | 全品种利好 |
| 区间震荡 | 品种自身驱动为主 |

---

## 第四层：通胀趋势

### 判定逻辑

```
CPI >4% 持续3个月 → 通胀失控 → 黄金利好
CPI <2% 持续3个月 → 通缩风险 → 黄金利空（现金为王）
PCE核心（美联储最看重）>2.5% → 美联储可能不降息
```

---

## 第五层：就业与增长

### 判定

```
失业率上升0.3%+ = 劳动力恶化 → 降息预期 → 黄金利好
JOLTS大幅下降 = 企业减少招聘 → 领先失业率6个月
GDP负增长 = 衰退 → 全品种重新评估
```

---

## 联动关系

### 黄金最敏感的组合

```
TIPS↓ + DXY↓ + VIX↑ = 黄金最佳环境
TIPS↑ + DXY↑ + VIX↓ = 黄金最差环境
```

### 对其他品种的传导

| 宏观变化 | 原油 | 铜 | 大豆 |
|---------|------|----|------|
| 美元走弱 | 利好 | 利好 | 利好 |
| 利率下降 | 中性偏多 | 利好 | 中性 |
| 衰退预期 | 利空（需求萎缩） | 利空 | 利空 |
| 通胀上升 | 温和利好 | 中性 | 温和利好 |

---

## 数据优先级（看哪个先不看哪个）

1  TIPS（实际利率）— 第一优先
2  10年国债 — 名义利率方向
3  FOMC概率 — 央行路径
4  美元指数 — 全品种锚
5  10Y盈亏平衡 — 通胀预期
6  CPI/PCE — 通胀验证
7  失业率 — 经济健康
8  GDP — 增长
9  收益率利差 — 衰退预警
10 VIX — 情绪
