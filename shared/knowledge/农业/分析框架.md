# 农业分析框架

---

## 核心逻辑：农业 ≠ 宏观

```
农业 = 天气驱动 + 供需平衡 + 地缘政策
```

**利率和美元影响远小于天气和USDA数据。**

---

## 大豆分析

### 第一优先：南美天气（12月-3月）

```
巴西/阿根廷干旱 → 大豆减产 → 中国转向美豆 → 美豆涨价
南美风调雨顺 → 南美增产压价
```

### 第二优先：美国天气（6月-8月生长季）

```
中西部干旱 → 大豆优良率↓ → 减产预期 → 看多
中西部降水正常 → 中性
```

### 第三优先：USDA优良率

```
优良率↓10%+ 连续两周 → 减产在发生 → 看多
优良率稳定 60-75% → 正常
```

### 第四优先：中国采购量（出口检验数据）

```
中国采购连增 → 需求旺盛 → 看多
中国转向南美 → 美豆需求减 → 看空
```

### 第五优先：大豆-玉米比价

```
大豆:玉米 > 2.5 → 明年种植大豆更划算 → 下季扩产 → 远月承压
```

## 玉米分析

### 第一优先：美国天气（6-8月授粉期）

```
7月授粉期高温干旱 → 单产受损 → 暴涨
夏季温和多雨 → 高产 → 承压
```

### 第二优先：乙醇需求

```
美国乙醇生产量 → 玉米35%用于乙醇 → 油价涨后玉米跟进
```

### 第三优先：USDA库存报告

```
季度库存 < 预期 → 供应紧张 → 看多
库存 > 预期 → 过剩 → 看空
```

---

## 小麦分析

### 第一优先：黑海局势

```
黑海供给占全球30% → 出口中断 → 全球短缺 → 暴涨
黑海通畅 → 供应充足 → 稳定
```

### 第二优先：美国冬小麦生长

```
冬小麦出苗期干旱 → 减产 → 看多
春季返青正常 → 中性
```

## 豆油/豆粕关系

```
大豆压榨 → 产出 豆油(19%) + 豆粕(79%)
```

| 因子 | 对豆油 | 对豆粕 |
|------|--------|--------|
| 原油上涨 | 利好（生物柴油） | 中性 |
| 生猪存栏增 | 中性 | 利好（饲料需求） |
| 中美关税 | 利空 | 利空 |

## 什么时候关注

| 情景 | 关注品种 |
|------|---------|
| 美国中西部30天无雨 | 大豆、玉米 |
| USDA优良率低于60% | 大豆、玉米 |
| 中国取消美豆订单 | 大豆利空 |
| 黑海冲突消息 | 小麦暴涨 |
| 巴西干旱新闻 | 关注大豆转向美豆 |
| COT大豆 > 90 | 警惕回调 |
