## USDA数据（全球最重要）

### 优良率

| 项目 | 内容 |
|------|------|
| 数据源 | USDA 作物生长报告（需手动采集） |
| 含义 | 生长状况评级（优+良占比） |
| 正常区间 | 60-75% |
| ✅ 看多 | 大幅低于去年同期（减产预期） |
| 🔻 看空 | 高于或等于去年同期（供应充足） |
| 发布时间 | 每周二凌晨（北京时间） |

### 出口检验

| 项目 | 内容 |
|------|------|
| 含义 | 每周向海外出口检验数量 |
| 关键 | 中国买家量是核心（大豆最敏感） |

### 种植面积与产量

| 项目 | 内容 |
|------|------|
| 含义 | USDA预测的种植面积和单产 |
| 核心报告 | 3月种植意向、6月种植面积、8-11月单产调整 |

### WASDE供需报告

| 项目 | 内容 |
|------|------|
| 含义 | 全球供需月度预测 |
| 关键数据 | 期末库存、产量预估、出口预估 |

## 与天气联动

| 天气事件 | 受影响品种 |
