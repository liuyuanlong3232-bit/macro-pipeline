## 怎么添加新内容

1. 参考 `模板.md` 中的结构
2. 新指标加在对应模块的 `数据字典.md` 末尾
3. 新分析方法加在对应模块的 `分析框架.md` 末尾
4. 改完提交，"增量更新，不要重写"
