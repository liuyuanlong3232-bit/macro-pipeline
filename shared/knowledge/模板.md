# 模板

## 数据字典怎么写

每个指标一个条目，格式：

```
### 指标中文名（代码）

| 项目 | 内容 |
|------|------|
| 数据源 | 从哪来的（FRED/Yahoo/CFTC...） |
| 含义 | 这个指标代表什么 |
| 正常区间 | 历史中枢是多少 |
| 看多信号 | 什么数值/方向是利好 |
| 看空信号 | 什么数值/方向是利空 |
| 极端值 | 什么情况需要高度关注 |
| 发布时间 | 数据多久更新一次 |
```

## 分析框架怎么写

按判断流程写，每步说清楚：

```
### 某个品种/领域的判断流程

优先级从高到低排列：

1️⃣ 第一层：最重要的是什么
   - 看哪个指标 → 什么方向代表利好/利空
   - 为什么排第一

2️⃣ 第二层：次要因素
   - ...

3️⃣ 第三层：辅助因素
   - ...
```
