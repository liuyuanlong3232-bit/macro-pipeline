# 天气分析框架

---

## 核心逻辑

```
天气 → 产量预期 → 期货定价
```

**天气是农业的第一驱动力，优先于USDA、COT、美元利率。**

---

## 分析流程

### 第一层：哪里的天气最值得看

```
当前生长季作物 → 主产地天气 → 降水量/温度异常
```

**季节性焦点：**

| 现在几月 | 盯哪里 | 盯什么 |
|---------|--------|--------|
| 12-2月 | 南美（巴西/阿根廷） | 大豆生长期降水 |
| 3-4月 | 美国中西部 | 玉米/大豆播种期降水 |
| 5-6月 | 美国中西部 | 出苗+生长温度 |
| **7月** | **美国中西部** | **玉米授粉（最敏感）** |
| 8月 | 美国中西部 | 大豆结荚+玉米灌浆 |
| 9-10月 | 美国中西部 | 收获期降雨（过多不利） |
| 11月 | 南美 | 大豆播种期降水 |

### 第二层：连续性最重要

```
一次干旱 → 不影响
两周干旱 → 轻微影响，关注
三周干旱+30天预报无雨 → 严重影响，准备做多
```
**连续性是判断严重程度的关键。**

### 第三层：USDA反应滞后于天气

```
天气恶化 → 作物受损 → 1-2周后USDA调降优良率 → 2-4周后USDA调降产量预估
```

**天气变化 → 期货立刻反应，USDA确认后加速。**

---

## 关键监测点

### 美国中西部生长季（4-9月）

```
监测要点：
- 30天累计降水 vs 历史同期
- 土壤湿度百分比
- 最高温度超35°C天数
- USDA每周优良率变化
```

### 南美生长季（11-3月）

```
监测要点：
- 巴西MT/PR/RS降水量
- 阿根廷大豆产区降水
- 如果连续3周降水<50% → 关注大豆
```

### 黑海地区（全年）

```
监测要点：
- 小麦关键期降水
- 冬季降雪覆盖（隔热防冻）
- 早霜/晚霜时间
```
