# 贵金属分析框架

---

## 黄金分析流程

### 第一优先：TIPS（实际利率）

```
TIPS↑（实际利率上升） → 持有黄金成本上升 → 黄金承压
TIPS↓（实际利率下降） → 持有黄金成本下降 → 黄金利好
```

**这是黄金最重要的驱动因子，优先级排在一切前面。**

### 第二优先：美元方向

```
DXY↑ → 黄金以美元计价，美指强黄金承压
DXY↓ → 弱美元提振黄金
```

### 第三优先：COT极端值

```
COT Index > 90 → 极端看多，多头拥挤，警惕回调
COT Index < 10 → 极端看空，反弹概率高
```

**COT极端值是最可靠的短期信号之一。**

### 第四优先：VIX避险情绪

```
VIX > 25 → 市场恐慌，避险资金进黄金
VIX < 15 → 风险偏好高，资金流向风险资产
```

### 第五优先：金银比

```
金银比 > 85 → 白银极度低估，仓位倾向白银
金银比 < 55 → 白银过热
```

## 黄金的"看好/看淡"综合判定

### 看好黄金

- [ ] TIPS 在下降通道或负值
- [ ] DXY 在走弱
- [ ] COT Index < 40（持仓不拥挤）
- [ ] VIX > 20 或在上升

**≥3项满足 = 较强看多信号**

### 看淡黄金

- [ ] TIPS 在上升通道且 >2%
- [ ] DXY 在走强
- [ ] COT Index > 80（多头拥挤）
- [ ] VIX < 15（太低，无避险需求）

**≥3项满足 = 较强看空信号**

## 白银分析流程

### 白银 = 黄金因子 + 工业因子

| 因子 | 看什么 | 利好白银条件 |
|------|--------|------------|
| 黄金引力 | 黄金自身方向 | 黄金上涨 |
| 金银比 | 相对估值 | 比值 > 75 |
| 工业需求 | 全球PMI、铜价 | 制造业扩张 |
| 太阳能 | 光伏装机数据 | 装机增长 |

### 白银的特殊性

- 波动是黄金的 **2倍**（涨得多跌得也多）
- 工业需求占白银消费 **50%+**（铜价相关性高）
- 光伏用银是新增长驱动（年增速10-15%）

---

## 什么时候该操作

| 情景 | 操作 |
|------|------|
| TIPS下降 + DXY下降 + COT < 50 | 黄金配置 |
| COT > 95 + 黄金连涨3周 | 警惕回调，设止损 |
| 金银比 > 85 | 白银优于黄金 |
| VIX突然从15冲到25 | 避险爆发，关注黄金 |
| DXY单周涨2%+ | 全贵金属品种承压 |

---

## 什么时候不该操作

| 情景 | 操作 |
|------|------|
| 所有指标中性 | 不做，等信号 |
| COT极端 + 方向不明确 | 等明确方向再动 |
| FOMC前3天 | 观望，会后出方向 |
| 行情单日涨跌 >3%（无事件驱动） | 可能是算法交易，等回归 |
