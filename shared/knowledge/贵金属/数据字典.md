## 现货/期货价格

### COMEX黄金期货（GC=F）

| 项目 | 内容 |
|------|------|
| 数据源 | Yahoo Finance → `yahoo_futures` |
| 含义 | COMEX黄金期货最近月合约 |
| 计价 | 美元/盎司 |
| 日内波动 | 通常 ±0.5-2% |
| 极端波动 | >3% 值得打预警 |
| DB列名 | 品種=COMEX黃金期貨 |

### COMEX白银期货（SI=F）

| 项目 | 内容 |
|------|------|
| 数据源 | Yahoo Finance → `yahoo_futures` |
| 含义 | COMEX白银期货 |
| 日内波动 | 通常 ±1-4% |
| ⚠️ | 波动是黄金的2倍，注意仓位 |
| DB列名 | 品種=COMEX白銀期貨 |

## COT持仓（关键信号）

### COT(26周)

| 项目 | 内容 |
|------|------|
| 数据源 | cotdata.net → `cotdata` 表 |
| 含义 | 投机持仓偏离度，0-100 |

| 区间 | 信号 | 操作参考 |
|------|------|---------|
| 90-100 | **极度看多**，警惕回调 | 减仓或设止损 |
| 60-90 | 偏多 | 持有 |
| 40-60 | 中性 | 观望 |
| 10-40 | 偏空 | 关注超卖 |
| 0-10 | **极度看空**，关注反弹 | 准备做多 |

### Z-Score

| 项目 | 内容 |
|------|------|
| 数据源 | cotdata.net |
| 含义 | 持仓偏离度标准差 |
| 极端值 | >2或<-2（显著偏离） |

### 其他 COT指标（cot_history表）

| 列名 | 含义 |
|------|------|
| managed_money_long / short | 管理基金持仓 |
| swap_long / short | 掉期交易商持仓 |
| producer_merchant_long / short | 生产商持仓 |

## 金银比

### 金银比 = 黄金价 / 白银价

| 项目 | 内容 |
|------|------|
| 数据源 | 计算得出 |
| 历史中枢 | 60-70 |
| 正常区间 | 50-85 |

| 区间 | 信号 | 操作参考 |
|------|------|---------|
| >85 | 白银极度低估 | 关注白银补涨 |
| 75-85 | 白银低估 | 白银优于黄金 |
| 60-75 | 正常 | 中性 |
| 50-60 | 白银高估 | 黄金优于白银 |
| <50 | 白银极度高估 | 小心过热 |

## VIX

### VIX恐慌指数

| 项目 | 内容 |
|------|------|
| 数据源 | CFTC TFF → `vix_data` |
| DB列名 | 品種=VIX |

| 区间 | 对黄金影响 |
|------|-----------|
| <12 | 风险偏好极高，黄金承压 |
| 12-20 | 正常，不影响 |
| 20-25 | 避险升温，黄金温和利好 |
| 25-35 | 高度恐慌，黄金利好 |
| >35 | 市场危机，黄金看涨但注意流动性 |
