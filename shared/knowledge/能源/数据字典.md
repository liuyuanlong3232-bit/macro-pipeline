## 现货/期货价格

### WTI原油期货（CL=F）

| 项目 | 内容 |
|------|------|
| 数据源 | Yahoo Finance → `yahoo_futures` |
| 含义 | 美国西德克萨斯轻质原油 |
| 正常区间 | $60-90 |
| 看多 | <$70（接近美国页岩油成本线） |
| 看空 | >$100（需求萎缩） |
| 极端值 | <$50或>$120 |
| DB列名 | 品種=WTI原油期貨 |

### Brent原油期货（BZ=F）

| 项目 | 内容 |
|------|------|
| 含义 | 国际基准原油 |
| 正常区间 | $65-95 |
| WTI-Brent价差 | 正常-$3到-$8 |
| DB列名 | 品種=Brent原油期貨 |

### 天然气 Henry Hub（NG=F）

| 项目 | 内容 |
|------|------|
| 含义 | 美国天然气基准 |
| 正常区间 | $2-5 |
| 极端值 | <$2（负利）或>$10 |
| DB列名 | 品種=天然氣期貨(Henry Hub) |

## EIA库存（核心基本面）

### 原油库存

| 项目 | 内容 |
|------|------|
| 数据源 | EIA API → `energy_history` |
| 正常季节 | 夏季去库、冬季累库 |
| ⚠️ 看多 | 库存降幅 > 预期 |
| 看空 | 库存增幅 > 预期 |
| 发布时间 | 每周三 22:30 北京时间 |
| DB列名 | crude_inventory |

### 汽油库存

| 项目 | 内容 |
|------|------|
| 含义 | 驾驶季核心指标 |
| 关键 | 5-9月美国驾驶旺季去库 > 预期利好原油 |
| DB列名 | gasoline_inventory |

### 天然气库存（AGSI+）

| 项目 | 内容 |
|------|------|
| 数据源 | AGSI+ API → `energy_history` |
| 含义 | 欧洲天然气库存 |
| 正常区间 | 冬季去库、夏季注气 |
| 极端值 | 填充率<20%（危机）或>90%（过剩） |
| DB列名 | gas_in_storage, working_capacity |

## 与宏观联动

| 宏观变化 | WTI | 天然气 |
|---------|-----|--------|
| 美元走弱 | 利好 | 中性 |
| 利率下降 | 中性偏多 | 中性 |
| 衰退预期 | 利空（需求减少） | 利空 |
| 冬季寒潮 | 中性 | 暴涨 |
| 飓风墨西哥湾 | 短期利多 | 利多 |
