# 能源分析框架

---

## 原油分析流程

### 第一优先：美元（宏观定价锚）

```
DXY↓ → 弱美元 ⇒ 以美元计价的原油涨价
DXY↑ → 强美元 ⇒ 原油承压
```

### 第二优先：EIA库存（供需基本面）

```
库存↓超预期 → 供给紧张 → 看多
库存↑超预期 → 供给过剩 → 看空
```

**EIA库存是原油最直接的供需信号，每周三必看。**

### 第三优先：WTI-Brent价差

```
价差扩张（<-$8）→ 美国供应过剩，WTI弱
价差收缩（>-$3）→ 墨西哥湾运输紧张，WTI强
```

### 第四优先：产量趋势

```
美油产量↑ → 长期利空（OPEC+份额被蚕食）
OPEC+减产 → 短期利多 → 看油价能否站稳
```

## 原油的"看好/看淡"综合判定

### 看好原油

- [ ] EIA库存连续下降
- [ ] DXY走弱
- [ ] 全球PMI > 50（制造业扩张→需求增大）
- [ ] OPEC+延长减产

**≥3项满足 = 较强看多**

### 看淡原油

- [ ] EIA库存连续上升
- [ ] DXY走强
- [ ] 全球经济减速
- [ ] US产量创历史新高

**≥3项满足 = 较强看空**

## 天然气分析

### 天然气 ≠ 原油

```
天然气 = 天气驱动 + 库存 + 产量
```

### 季节规律

| 季节 | 需求 | 库存 |
|------|------|------|
| 冬季（11-3月） | 🔺取暖高峰 | 去库 |
| 春季（4-5月） | 🔻淡季 | 注气 |
| 夏季（6-8月） | 🔺空调发电 | 稳定 |
| 秋季（9-10月） | 🔻淡季 | 注气 |

### 判定

```
寒潮预报 → 天然气暴涨（天气是最强短期驱动）
库存低位 + 冬季来临 → 供应担忧 → 看多
产量创新高 + 温和天气 → 供应过剩 → 看空
```

## Brent-WTI价差是什么信号

| 价差 | 信号 |
|------|------|
| $3-$5 | 正常 |
| >$8（WTI远低于Brent） | 美国供应过剩或运输瓶颈 |
| <$2（WTI接近Brent） | 墨西哥湾供应紧张 |

## 什么时候该操作

| 情景 | 操作 |
|------|------|
| EIA库存超预期下降 | 短期利多 |
| 冬季寒潮天气预报 | 关注天然气 |
| 钻井数量连续下降 | 关注原油供给 |
| OPEC+会议前一周 | 观望，会后再动 |

---

## 什么时候不该操作

| 情景 | 操作 |
|------|------|
| EIA库存符合预期 | 不操作 |
| 地缘消息激发短线 | 等消息消化 |
| 假期（流动性薄） | 不操作 |
