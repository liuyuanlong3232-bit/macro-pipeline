# 数据合规发布规则

## 三类数据源合规等级

### 🟢 直接可用（官方API/官方CSV）
**规则：** 可以直接使用，但推荐转化后呈现
**示例：** USDA NASS API、Open-Meteo天气API、Tushare官方API、国家统计局数据
**呈现方式：** ✅ 趋势+图表+来源标注
**注意：** Tushare期货数据属于商业API，买了Token即可用，但发的是分析和趋势，不是原始数据表

### 🟡 可呈现（公开网页/PDF爬取）
**规则：** 必须转化后呈现，不能贴原文
**示例：** USDA优良率（Cornell公开PDF）、USDA出口检验（AMS政府TXT）、美湾港口库存（CSV下载）
**呈现方式：** ⚠️ 仅趋势描述+图表，不露原始数据
**法律依据：** 美国政府数据（USDA）属于公共领域（Public Domain），数据本身不受版权保护

### 🔴 不可呈现（付费数据/版权数据/个人数据）
**规则：** 不采集不使用不呈现
**示例：** 任何商业数据库（Bloomberg/Refinitiv）、需要付费的API、个人/企业内部数据

## 发布三原则

### 1. 数据转化（不露原始数据）
```
❌ "USDA报告显示大豆优良率72%，小麦优良率65%"（直接引用原始数据）
✅ "大豆优良率处于近5年同期高位"（趋势描述）
```

### 2. 趋势而非绝对值
```
❌ "最新价格 4,230元/吨"
✅ "价格处于近3个月震荡区间上沿"
```

### 3. 分析而非数据表
```
❌ 贴出USDA报告原文的产量数据表
✅ 输出分析结论：USDA月报显示大豆产量预期上调
```

## 中国数据特殊规则

| 数据 | 合规 | 能发什么 | 不能发什么 |
|------|------|---------|-----------|
| Tushare中国期货 | ✅ 商业API可用 | 趋势分析（"多头增仓"、"持仓量放大"） | 原始持仓手数数据表 |
| 期货价格 | ✅ 公开行情 | 趋势描述（"震荡区间"） | 绝对价格预测 |
| CNGOIC压榨率 | 🟠 需确认TOS | 转化后可用，不贴原文 | PDF原文/原始数据 |
| 国家统计局数据 | ✅ 政府公开 | 趋势+图表 | 国家秘密级数据 |
| 农业农村部数据 | ✅ 政府公开 | 分析+结论 | 敏感信息 |

## 公众号发布自查清单

```
□ 标题是否客观，是否带情绪词汇
□ 是否涉及具体价格预测
□ 是否涉及交易建议
□ 是否贴了原始数据表/原文
□ 是否有免责声明
□ 是否标注数据来源
□ 内容是否属于农业数据分析而非金融投资
□ 是否有涉及未公开信息的嫌疑
```

```
【免责声明】
本报告数据均来源于公开渠道（USDA、气象局、期货交易所等），

报告中的分析和判断不构成任何投资建议，
读者应独立判断并承担相应风险。
```
